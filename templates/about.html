Setup

  • Tensorfow GPU software requirements
    NVIDIA GPU drivers —CUDA 10.0 requires 418.x or higher
    CUDA Toolkit —TensorFlow supports CUDA 10.0
    CUPTI ships with the CUDA Toolkit
    cuDNN SDK (>= 7.6)
  • create & active environment
    conda env create -v -f tensorflow-gpu.yml
  • name: inceptionModel
    dependencies:
    - python=3.7
    - pip>=19.0
    - jupyter
    - tensorflow-gpu=1.15.0
    - pandas=0.25.3
    - pandas-datareader
    - matplotlib=3.0.3
    - pillow=7.0.0
    - tqdm
    - requests
    - h5py
    - flask
    conda activate inceptionModel
  • Test to see if tensorflow is working
    (tensorflow)C:\Users\username> python
    >>> import tensorflow as tf
    >>> print (tf.__version__)
    (should get 1.15.0)
    >>> quit()
    close command prompt
